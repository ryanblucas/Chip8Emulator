# Chip8Emulator
I wrote this emulator primarily to help me transition from C development to C++. Hence, the goal of this program is not to create a 100% efficient and faithful CHIP-8 emulator but for me to learn certain idioms and classes/objects that are necessary for professional C++ development.
There are some issues I'm aware of. Not all quirks work right/are configurable and there is some flickering.
